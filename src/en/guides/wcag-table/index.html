Introduction

The searchable WCAG table is a list of WCAG 2.1 AA testing techniques that can be used to test accessibility of websites. It is a useful reference for developers and designers to check the accessibility of their websites. It can also be a useful reference for users to check the accessibility of their websites.

The content of this table was generated by the Digital Accessibility Unit of Accessibility, Accommodations, and Adaptive Computer Technology Program of Shared Services Canada using the techniques and examples listed in the WCAG 2.1 documentation.
